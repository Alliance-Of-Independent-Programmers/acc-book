### Создание виртуального окружения
Чтобы начать работу с проектом, нужно создать виртуальное окружение 
python3 + node.js, для этого необходимо в каталоге проекта выполнить команды:
```shell
python3 -m venv .venv
source .venv/bin/activate
pip install nodeenv 
nodeenv -p
npm install -g yarn 
cd ./frontend 
yarn install

```
